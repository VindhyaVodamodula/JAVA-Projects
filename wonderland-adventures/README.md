# Wonderland Adventures

Wonderland Adventures is a text-based adventure game built in Java and inspired by the imaginative style of Alice in Wonderland. The player explores themed rooms, solves puzzles, collects items, and progresses through the game world to reach the final objective.

This project is especially meaningful to me because it combines technical learning with creativity. While building it, I explored ideas for gameplay, progression, and story design, and I see it as a project with future potential for visual expansion.

## Project Highlights

- Text-based adventure gameplay
- Multiple rooms and world navigation
- Inventory system
- Puzzle-solving mechanics
- Structured game flow using multiple Java classes
- External data file support
- Supporting design, testing, and reflection documentation

## Technologies Used

- Java
- Object-oriented programming
- Multiple class design
- File handling / data loading
- Conditional logic and loops
- Interactive command-line interface

## What I Learned

Through this project, I strengthened my understanding of:

- designing a larger Java program across multiple files
- organizing code using classes and responsibilities
- building game logic and player progression
- planning software using design and test documents
- thinking creatively about future software improvements

This project also includes:

## Future Direction

I plan to continue improving this project by exploring ways to make it more visual and interactive while keeping the original game concept and structure.
